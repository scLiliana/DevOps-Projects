## Database Tier: RDS MySQL

### Crear DB Subnet Group

#### Recuperar ID de subnets
**Pre-requisitos:** `$VPC_PRIMARY_ID`
```bash

SUBNET_DB_1A_ID=$(aws ec2 describe-subnets \
    --filters "Name=vpc-id,Values=$VPC_PRIMARY_ID" \
              "Name=tag:Name,Values=DBSubnet-1a" \
    --query "Subnets[0].SubnetId" --output text --region us-east-1)

SUBNET_DB_1B_ID=$(aws ec2 describe-subnets \
    --filters "Name=vpc-id,Values=$VPC_PRIMARY_ID" \
              "Name=tag:Name,Values=DBSubnet-1b" \
    --query "Subnets[0].SubnetId" --output text --region us-east-1)
    
```

#### Crear DB Subnet Group
**Pre-requisitos:** `$SUBNET_DB_1A_ID, $SUBNET_DB_1B_ID`
```bash

aws rds create-db-subnet-group \
    --db-subnet-group-name mydbsubnetgroup \
    --db-subnet-group-description "Subnet group for MySQL RDS" \
    --subnet-ids $SUBNET_DB_1A_ID $SUBNET_DB_1B_ID
echo "DB Subnet Group creado"
```

## Crear instancia RDS Multi-AZ

**Pre-requisitos:** `$SG_DATABASE_ID`
```bash
# Crear instancia MySQL

aws rds create-db-instance \
    --db-instance-identifier prod-mysql \
    --db-instance-class db.t3.micro \
    --engine mysql \
    --engine-version 8.0 \
    --master-username admin \
    --master-user-password "TuPasswordSegura123!" \
    --allocated-storage 20 \
    --multi-az \
    --vpc-security-group-ids $SG_DATABASE_ID \
    --db-subnet-group-name mydbsubnetgroup \
    --backup-retention-period 7 \
    --no-publicly-accessible \
    --tags Key=Name,Value=ProdMySQL
```

```bash
# Esperar hasta que esté disponible (~10-15 min)

aws rds wait db-instance-available \
    --db-instance-identifier prod-mysql \
    --region us-east-1
echo "RDS disponible"

```

**Pre-requisitos:** [[3.Función para guardar el variables en un documento]]
```bash
# Obtener y guardar el endpoint

RDS_ENDPOINT=$(aws rds describe-db-instances \
    --db-instance-identifier prod-mysql \
    --query "DBInstances[0].Endpoint.Address" \
    --output text --region us-east-1)

save_id "RDS_ENDPOINT" "$RDS_ENDPOINT"
```
