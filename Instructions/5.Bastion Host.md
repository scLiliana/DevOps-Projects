# Bastion host

**Pre-requisitos:** [[3.Función para guardar el variables en un documento]] 

## Obtener Mi IP Pública

```bash
# Detectar tu IP pública automáticamente

MY_IP=$(curl -s https://checkip.amazonaws.com)/32

echo "Tu IP pública detectada: $MY_IP"

save_id "MY_IP" "$MY_IP"

```

## Crear Security Group para Bastion (SSH solo desde tu IP)

**Pre-requisitos:** ```$VPC_PRIMARY_ID````
```bash
# Crear Security group

SG_BASTION_ID=$(aws ec2 create-security-group \
    --group-name BastionSG \
    --description "SSH access to Bastion Host" \
    --vpc-id $VPC_PRIMARY_ID \
    --query "GroupId" \
    --output text \
    --region us-east-1)
    
save_id "SG_BASTION_ID" "$SG_BASTION_ID"

```

**Pre-requisitos:** ```$SG_BASTION_ID, $MY_IP```
```bash
# Permitir acceso SSH de mi IP a Bastion

aws ec2 authorize-security-group-ingress \
    --group-id $SG_BASTION_ID \
    --protocol tcp \
    --port 22 \
    --cidr $MY_IP \
    --region us-east-1
```

## Iniciar instancia de Bastion

**Pre-requisitos:**```$GLOBAL_AMI, $SG_BASTION_ID, $SUBNET_PUBLIC_1A_ID```
```bash

INSTANCE_BASTION_ID=$(aws ec2 run-instances \
    --image-id $GLOBAL_AMI \
    --count 1 \
    --instance-type t3.micro \
    --key-name DevOpsKeyPair \
    --security-group-ids $SG_BASTION_ID \
    --subnet-id $SUBNET_PUBLIC_1A_ID \
    --associate-public-ip-address \
    --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=BastionHost}]' \
    --query "Instances[0].InstanceId" \
    --output text \
    --region us-east-1)
    
save_id "INSTANCE_BASTION_ID" "$INSTANCE_BASTION_ID"

```

**Pre-requisitos:**```$INSTANCE_BASTION_ID```
```bash
# Esperar a que este corriendo

aws ec2 wait instance-running \
	--instance-ids $INSTANCE_BASTION_ID \
    --region us-east-1
	
```

## Recuperar IP Bastion

**Pre-requisitos:**```$INSTANCE_BASTION_ID```
```bash

IP_BASTION=$(aws ec2 describe-instances \
    --instance-ids $INSTANCE_BASTION_ID \
    --query "Reservations[0].Instances[0].PublicIpAddress" \
    --output text --region us-east-1)
    
save_id "IP_BASTION" "$IP_BASTION"

```
