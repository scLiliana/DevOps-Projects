# DevOps-Proyect-01
---

Este proyecto demuestra la implementación de una aplicación web Java de nivel de producción utilizando la robusta arquitectura de tres niveles de AWS. La implementación sigue las mejores prácticas nativas de la nube, lo que garantiza alta disponibilidad, escalabilidad y seguridad en todos los niveles de la aplicación.
