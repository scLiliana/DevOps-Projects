# VPC y Networking

## Creación de las VPCs

```bash
# Crear la VPC primaria y guardar su ID

VPC_PRIMARY_ID=$(aws ec2 create-vpc \
    --cidr-block 192.168.0.0/16 \
    --tag-specifications 'ResourceType=vpc,Tags=[{Key=Name,Value=PrimaryVPC}]' \
    --query "Vpc.VpcId" --output text \
    --region us-east-1)
    
save_id "VPC_PRIMARY_ID" "$VPC_PRIMARY_ID"
```

```bash
# Crear la VPC secundaria y guardar su ID

VPC_SECONDARY_ID=$(aws ec2 create-vpc \
    --cidr-block 172.32.0.0/16 \
    --tag-specifications 'ResourceType=vpc,Tags=[{Key=Name,Value=SecondaryVPC}]' \
    --query "Vpc.VpcId" --output text --region us-east-1)
    
save_id "VPC_SECONDARY_ID" "$VPC_SECONDARY_ID"

```

## Creación del Internet Gateway

```bash
# Crear el IGW a la VPC principal

IGW_ID=$(aws ec2 create-internet-gateway \
    --tag-specifications 'ResourceType=internet-gateway,Tags=[{Key=Name,Value=PrimaryIGW}]' \
    --query "InternetGateway.InternetGatewayId" \
    --output text \
    --region us-east-1)
    
save_id "IGW_ID" "$IGW_ID"
```
**Pre-requisitos**:  ```$VPC_PRIMARY_ID, $IGW_ID```
```bash
# Adjuntar el IGW a la VPC principal

aws ec2 attach-internet-gateway \
    --vpc-id $VPC_PRIMARY_ID \
    --internet-gateway-id $IGW_ID \
    --region us-east-1
```

## Creación de NAT Gateways

### Elastic IPs

```bash
# Elastic IP para NAT AZ-1a

EIP_1A_ID=$(aws ec2 allocate-address \
    --domain vpc \
    --query "AllocationId" \
    --output text \
    --region us-east-1)
    
save_id "EIP_1A_ID" "$EIP_1A_ID"
```

```bash
# Elastic IP para NAT AZ-1b

EIP_1B_ID=$(aws ec2 allocate-address \
    --domain vpc \
    --query "AllocationId" \
    --output text \
    --region us-east-1)
    
save_id "EIP_1B_ID" "$EIP_1B_ID"
```

### NAT Gateways

**Pre-requisitos**:  ```$SUBNET_PUBLIC_1A_ID, $EIP_1A_ID```
```bash 
# Crear NAT Gateway en subnet pública AZ-1a

NAT_1A_ID=$(aws ec2 create-nat-gateway \
    --subnet-id $SUBNET_PUBLIC_1A_ID \
    --allocation-id $EIP_1A_ID \
    --tag-specifications 'ResourceType=natgateway,Tags=[{Key=Name,Value=NAT-1a}]' \
    --query "NatGateway.NatGatewayId" \
    --output text 
    --region us-east-1)
    
save_id "NAT_1A_ID" "$NAT_1A_ID"
```

**Pre-requisitos**:  ```$SUBNET_PUBLIC_1B_ID, $EIP_1B_ID```
```bash 
# Crear NAT Gateway en subnet pública AZ-1b

NAT_1B_ID=$(aws ec2 create-nat-gateway \
    --subnet-id $SUBNET_PUBLIC_1B_ID \
    --allocation-id $EIP_1B_ID \
    --tag-specifications 'ResourceType=natgateway,Tags=[{Key=Name,Value=NAT-1b}]' \
    --query "NatGateway.NatGatewayId" \
    --output text 
    --region us-east-1)
    
save_id "NAT_1B_ID" "$NAT_1B_ID"
```

```bash
# ⚠️ Espera ~2 min hasta que el estado sea "available"

aws ec2 wait nat-gateway-available \
--nat-gateway-ids $NAT_1A_ID $NAT_1B_ID \
--region us-east-1

echo "✅ NAT Gateways listos"
```

## Creación de Route Tables

### Route Table Pública 

**Pre-requisitos**:  ```$VPC_PRIMARY_ID```
```bash
# ── Route Table PÚBLICA (apunta al IGW) ─────────────────────

RT_PUBLIC_ID=$(aws ec2 create-route-table \
    --vpc-id $VPC_PRIMARY_ID \
    --tag-specifications 'ResourceType=route-table,Tags=[{Key=Name,Value=PublicRT}]' \
    --query "RouteTable.RouteTableId" \
    --output text 
    --region us-east-1)
    
save_id "RT_PUBLIC_ID" "$RT_PUBLIC_ID"

```

**Pre-requisitos**:  ```$RT_PUBLIC_ID, $IGW_ID, $SUBNET_PUBLIC_1A_ID, $SUBNET_PUBLIC_1B_ID```
```bash
# Crear la ruta
aws ec2 create-route \
    --route-table-id $RT_PUBLIC_ID \
    --destination-cidr-block 0.0.0.0/0 \
    --gateway-id $IGW_ID

# Asociar a las dos subnets públicas
aws ec2 associate-route-table \
--route-table-id $RT_PUBLIC_ID \
--subnet-id $SUBNET_PUBLIC_1A_ID

aws ec2 associate-route-table \
--route-table-id $RT_PUBLIC_ID \
--subnet-id $SUBNET_PUBLIC_1B_ID

```

## Creación del Transit Gateway

Transit Gateway (comunicación entre VPCs)

> ℹ️ **El Transit Gateway y la VPC Secundaria son opcionales para este proyecto.** La arquitectura de 3 tiers funciona completamente con una sola VPC. Solo lo necesitas si quieres conectar múltiples VPCs entre sí.

```bash
# Generación del Transit Gateway

TGW_ID=$(aws ec2 create-transit-gateway \
    --description "TGW para comunicacion entre VPCs" \
    --tag-specifications 'ResourceType=transit-gateway,Tags=[{Key=Name,Value=MainTGW}]' \
    --query "TransitGateway.TransitGatewayId" \
    --output text \
    --region us-east-1)
    
save_id "TGW_ID" "$TGW_ID"

```

**Pre-requisitos**:  ```$TGW_ID, $VPC_PRIMARY_ID, $SUBNET_PRIVATE_1A_ID, $SUBNET_PRIVATE_1B_ID```
```bash
# Adjuntar VPC Principal (suficiente para este proyecto)

aws ec2 create-transit-gateway-vpc-attachment \
    --transit-gateway-id $TGW_ID \
    --vpc-id $VPC_PRIMARY_ID \
    --subnet-ids $SUBNET_PRIVATE_1A_ID $SUBNET_PRIVATE_1B_ID
```

Si además quieres adjuntar la VPC Secundaria, primero crea una subnet dentro de ella:

**Pre-requisitos**:  ```$VPC_SECONDARY_ID```
```bash
# Crear subnet privada en la VPC Secundaria

SUBNET_SECONDARY_PRIVATE_ID=$(aws ec2 create-subnet \
    --vpc-id $VPC_SECONDARY_ID \
    --cidr-block 172.32.1.0/24 \
    --availability-zone us-east-1a \
    --tag-specifications 'ResourceType=subnet,Tags=[{Key=Name,Value=SecondaryPrivateSubnet}]' \
    --query "Subnet.SubnetId" \
    --output text \
    --region us-east-1)
    
save_id "SUBNET_SECONDARY_PRIVATE_ID" "$SUBNET_SECONDARY_PRIVATE_ID"

```

**Pre-requisitos**:  ```$VPC_SECONDARY_ID, $TGW_ID, $SUBNET_SECONDARY_PRIVATE_ID```
```bash
# Adjuntar VPC Secundaria al Transit Gateway

aws ec2 create-transit-gateway-vpc-attachment \
    --transit-gateway-id $TGW_ID \
    --vpc-id $VPC_SECONDARY_ID \
    --subnet-ids $SUBNET_SECONDARY_PRIVATE_ID
```
